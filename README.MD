# Интернет-магазин - Fullstack приложение

Веб-приложение для управления интернет-магазином с системой аутентификации JWT, refresh-токенами и ролевой моделью доступа (RBAC).

## Технологии

- **Backend**: Node.js, Express, bcrypt, jsonwebtoken
- **Frontend**: React, Vite, React Router, Axios, Bootstrap

## Функциональность

- Регистрация и вход пользователей
- JWT аутентификация (access + refresh токены)
- Автоматическое обновление токенов
- Ролевая модель (RBAC):
  - **Гость** - только регистрация/вход
  - **Пользователь** - просмотр товаров
  - **Продавец** - создание и редактирование товаров
  - **Администратор** - полный доступ + управление пользователями
- CRUD операции с товарами
- Управление пользователями (только для админа)
